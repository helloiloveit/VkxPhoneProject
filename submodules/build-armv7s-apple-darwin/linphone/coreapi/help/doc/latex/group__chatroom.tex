\subsection{Detailed Description}
{\bfseries  Exchanging text messages} \par
 Messages are sent using \doxyref{Linphone\-Chat\-Room}{p.}{group__chatroom_ga37902f3fc715b7f6451c241d0efd0299} object. First step is to create a \doxyref{chat room }{p.}{group__chatroom_gac1358c912ecbf68be76ab38c3a3618d8} from a peer sip uri. 
\begin{DoxyCode}
LinphoneChatRoom* chat\_room = linphone_core_create_chat_room(lc,\textcolor{stringliteral}{"sip:joe@sip.linphone.org"});
\end{DoxyCode}


\par
Once created, messages are sent using function \doxyref{linphone\-\_\-chat\-\_\-room\-\_\-send\-\_\-message()}{p.}{group__chatroom_ga86dd4ea02757c7a21f50863c370f9c10} . 
\begin{DoxyCode}
linphone_chat_room_send_message(chat\_room,\textcolor{stringliteral}{"Hello world"}); \textcolor{comment}{/*sending message*/}
\end{DoxyCode}
 \par
Incoming message are received from call back Linphone\-Core\-V\-Table.\-text\-\_\-received 
\begin{DoxyCode}
\textcolor{keywordtype}{void} text\_received(LinphoneCore *lc, LinphoneChatRoom *room, \textcolor{keyword}{const} 
      LinphoneAddress *from, \textcolor{keyword}{const} \textcolor{keywordtype}{char} *message) \{
        printf(\textcolor{stringliteral}{" Message [%s] received from [%s] \(\backslash\)n"},message,
      linphone_address_as_string (from));
\}
\end{DoxyCode}
 \par
 A complete tutorial can be found at \-: \doxyref{Chat room tutorial}{p.}{group__chatroom__tuto}
